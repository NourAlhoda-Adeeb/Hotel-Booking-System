AN Hotel Booking System

Requirements:
- PHP 8.1+
- MySQL
- Composer
- XAMPP

Installation:
1. Extract the project
2. Create database: hotel_db
3. Import database/database/hotel_db.sql
4. Configure .env file
5. Run: php spark serve
6. Open: http://localhost:8080

Project Structure 
- app/ → Controllers, Models, Views
- public/ → CSS, JS, Images , uploads
- database_backup/ → SQL Database **hotel_db.sql**
